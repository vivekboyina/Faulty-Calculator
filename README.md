## 🚀 How to Use

1. Open the live project:  
   🔗 [View](https://vivekboyina.github.io/Faulty-Calculator/)  

2. The calculator will automatically start using popup prompts  

3. Follow the steps:
   - Enter the first number  
   - Enter the second number  
   - Choose an operator (`+`, `-`, `*`, `/`, `^`)  

4. The result will be shown in an alert  

---

## 🔄 Play Again

To use the calculator again, simply:

- 🔄 Refresh the page  

Each refresh runs the calculator again with a new random chance of faulty output.

---

## ⚠️ Note

- Most of the time results are correct  
- Sometimes (~10%), the result will be intentionally correct 😄  

---

## 💡 Tip

- Open Developer Tools (`F12`) → Console tab
